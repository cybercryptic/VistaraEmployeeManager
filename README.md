# Vistara Employee Manager 3.0
VEM 3.0 is a basic CRUD Application with MVC pattern.

## Tech
### Frontend
  1. HTML
  2. CSS
  3. JS
  4. Bootstrap 5
  5. JSP
### Backend
  1. Java
  2. Servlet
  3. MySQL
  4. HikariCP
  5. Hibernate

## How to use
1. Download or Clone
2. Run with your preferred server

No auth is implemented, just run.
